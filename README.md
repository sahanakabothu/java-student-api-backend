# Java Student REST API (Backend Style)

## Description
This project simulates a backend service for managing student data using Java.

## Features
- Add Student (Create)
- View Students (Read)
- Update Student
- Delete Student

## Concepts Used
- OOP (Classes & Objects)
- Service Layer Design
- CRUD Operations
- Java Collections

## Note
This is a backend-style simulation of REST API logic without using frameworks like Spring Boot.
